# **Algorytmy i Złożoność**

Aplikacja okienkowa w Pythonie umożliwiająca:

* generowanie określonej ilości losowych liczb
* wykonywanie podstawowych obliczeń matematycznych:
  * średnia arytmetyczna
  * średnia harmoniczna
  * średnia geometryczna
  * odchylenie standardowe
  * moda
  * rozstęp
  * mediana

Całość została przedstawiona w formie aplikacji okienkowej z użyciem biblioteki TKinter.
